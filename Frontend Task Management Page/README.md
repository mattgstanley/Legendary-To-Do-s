# Frontend - Task Management System

React frontend for the Construction Task Management System.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The build output will be in the `dist` directory.

## Environment Variables

Create a `.env` file with:

```
VITE_API_URL=
```

Leave empty for production (will use relative URLs). For local development, set to `http://localhost:3000`.
